IDEA常用快捷键For Mac

Posted on 2020-04-01 In 基础
Symbols count in article: 660 Reading time ≈ 1 mins.

查看

  • Command + 鼠标放在代码上 显示代码简要信息
  • Command + F1 在错误或警告处显示具体描述信息
  • Control + J 显示当前位置的变量、方法的 Documentation 内容

自动生成

  • Command + N, Control + Enter, Control + N 生成代码（getter、setter、构造函数、hashCode/equals,toString）
  • Control + O 覆盖方法（重写父类方法）
  • Control + I 实现方法（实现接口中的方法）

编辑

  • Command + Option + T 包围代码（使用if..else, try..catch, for, synchronized等包围选中的代码）
  • Command + / 注释/取消注释与行注释
  • Command + Option + / 注释/取消注释与块注释
  • Option + 方向键上 连续选中代码块，注意是按代码层级选择代码块
  • Option + 方向键下 减少当前选中的代码块
  • Command + X 剪切当前行或选定的块到剪贴板，如果是单行操作，可以无需选中
  • Command + C 复制当前行或选定的块到剪贴板
  • Command + Shift + V 从最近的缓冲区粘贴
  • Command + Option + L 格式化代码
  • Control + Option + O 优化import
  • Option + 方向键左/右 按单词跳过，向前或向后
  • Command + 方向键左/右 跳转到行
